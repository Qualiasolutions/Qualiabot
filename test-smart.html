QualiaBot Smart Widget Test

This is a test page for the QualiaBot Smart Widget, which uses local data from sitemapData.json to provide faster responses with fewer API calls.

Testing Features

  • Click the chat button in the bottom right corner to open the widget
  • The widget will try to answer questions about Qualia Solutions using local data first
  • It will only call the Perplexity API if it can't find a suitable answer locally
  • Try asking questions like "What services do you offer?" or "How can I contact you?"

Benefits of the Smart Widget

  • Reduced API usage and costs
  • Faster responses for common questions
  • Less reliance on external services
  • Automatic citations to relevant pages
